# Big Data Analytics Pipeline for Amazon Industrial and Scientific Reviews

Technologies:
- Docker
- Hadoop HDFS
- YARN
- Apache Spark
- Spark SQL
- Hive
- PySpark MLlib

Dataset:
Amazon Reviews 2023 – Industrial and Scientific Reviews

Model:
Random Forest Classifier

Results:
- Accuracy: 94.74%
- F1 Score: 0.9218
